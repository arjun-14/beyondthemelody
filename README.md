# Beyond the Melody
Flutter application that can detect emotion and recommend a playlist as per the detected emotion.

![](working.gif)



## Install
1. Obtain azure and youtube api keys and add them inside ```keys.dart``` under ```utilities``` in ```lib```.
2. ```$ flutter run```

## Emotions detected
- happiness
- sadness
- neutral
- anger
- contempt
- disgust
- fear
- surprise
